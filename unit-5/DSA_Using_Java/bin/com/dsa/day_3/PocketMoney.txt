Description

Akash's father gave him Rs X and tell him to go to market and buy fruits for Rs Y and the remaining money is his pocket money, print the pocket money Akash get.


Input
Input Format

First-line contains T, no of test cases.

First-line of each test case contains two integers a and b.

Constraints

1 <= T <= 10

1 <= a, b <= 10^4


Output
For each test case print the pocket money of Akash, in a new line.


Sample Input 1 

2
5 4
8 2
Sample Output 1

1
6
Hint

5 - 4 = 1

8 - 2 = 6
